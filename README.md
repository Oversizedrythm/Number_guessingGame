Guess the Number Game

A simple, interactive number guessing game built with HTML, CSS, and JavaScript.

**How to Play**

1. The computer will think of a random number between 1 and 20.

2. You have 4 attempts to guess the number.

3. Enter your guess into the input box and click the "Guess" button.

4. The game will tell you if your guess is "Too high!" or "Too low!".

5. If you guess the number, you win!

6. If you run out of guesses, the game will reveal the secret number.

7. Click "Play Again?" to start a new game.

**Technologies Used**

~ HTML: For the basic structure and content of the game.

~ CSS: For all styling, layout, and responsiveness.

~ JavaScript: For the game logic, including generating the random number and checking guesses.
